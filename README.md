# Crud de Usuarios BD GYM
## 🚀 Funcionalidades del módulo

- ✅ **Insertar usuario**
- 🔍 **Consultar todos los usuarios**
- ✏️ **Actualizar usuario**
- 🗑️ **Eliminar usuario**
- 💾 Conexión a base de datos con **JDBC**
- 📦 Proyecto estructurado con **Maven**
- 🧩 Versionamiento con **Git y GitHub**
